# Task CLI - Command Line Task Manager

## Overview
Task CLI is a command-line task management application built with Node.js. It allows users to create, list, complete, and delete tasks with persistent storage using JSON files.

## Features
- ✅ Add new tasks
- 📋 List all tasks or filter by status
- ✔️ Mark tasks as completed
- 🗑️ Delete tasks with confirmation
- 📊 View task statistics
- 💾 Persistent storage (tasks.json)
- 🎨 Color-coded console output
- 🔄 Auto-incrementing task IDs
